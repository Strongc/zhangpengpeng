7 Edwin

This chapter describes how to start Edwin, the MIT/GNU Scheme text editor. Edwin is very similar to GNU Emacs—you should refer to the GNU Emacs manual for information about Edwin's commands and key bindings—except that Edwin's extension language is MIT/GNU Scheme, while GNU Emacs extensions are written in Emacs Lisp. This manual does not discuss customization of Edwin.
